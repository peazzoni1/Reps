How to Export to PNG

  1. Open logo.svg in your browser (or view above)
  2. Use one of these methods to convert to PNG at 1024×1024:
    • Online: Upload logo.svg to CloudConvert and set size to 1024×1024
    • macOS: Open in Preview → File → Export → Set width to 1024 → Save as PNG
    • Command line (if you have ImageMagick):
      convert -background none -resize 1024x1024 logo.svg icon.png
  3. Replace the following files:
    • assets/icon.png (dark background version)
    • assets/adaptive-icon.png (for Android)
    • assets/splash-icon.png (for splash screen)

Tips

  • The dark background version (#1f2e4f) matches your app's color scheme
  • For Android adaptive icons, you might want to use a transparent or white background
  • Test both versions to see which looks better on different backgrounds
  • Make sure to export at exactly 1024×1024 pixels for best quality
